# CUMBRE — элитные экспедиции в аргентинских Андах

Премиальный лендинг для экспедиционной компании. Next.js 14 (App Router) + TailwindCSS +
Framer Motion + Lucide Icons. Без бэкенда — форма заявки работает только на клиенте.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

Продакшн-сборка:

```bash
npm run build
npm run start
```

## Структура секций (app/page.tsx)

1. **Navbar** — липкая навигация, меняет фон при скролле
2. **Hero** — полноэкранное видео (сноу/дрон), заголовок «Вершина начинается за 60 дней до экспедиции»
3. **Philosophy** — короткий тезис позиционирования
4. **WhyDifferent** — 8 карточек: чем компания отличается от обычного гида
5. **Expeditions** — 5 вершин (Аконкагуа, Охос-дель-Саладо, Ланин, Серро-Плата, Писис) с высотой, сложностью, длительностью, сезоном, местами и ценой
6. **Timeline** — 7-шаговый цикл подготовки (оценка формы → сила → выносливость → горная специфика → снаряжение → Аргентина → вершина)
7. **Team** — 2 профиля: горный гид и тренер, со статистикой
8. **Audience** — 6 карточек «кому это подходит»
9. **ApplicationProcess** — 6 шагов отбора + блок доверия про право отклонить заявку
10. **FAQ** — 17 вопросов-ответов в аккордеоне
11. **Contact** — форма заявки (имя, страна, возраст, телефон, email, вершина, уровень активности, опыт, сообщение)
12. **Footer** — минимальный: Instagram, WhatsApp, Telegram, Email

## Дизайн-токены

- **Цвет**: `obsidian #0A0C0F`, `ash #15181D`, `graphite #1F242B`, `stone #4A5560`,
  `mist #8B96A1`, `glacier #3E6C8E`, `glacier-light #6FA0C2`, `snow #F6F8FA`.
- **Шрифты**: PT Sans Narrow (display, заголовки в верхнем регистре — узкие пропорции,
  нативная поддержка кириллицы), Inter (текст), JetBrains Mono (цифры, показатели, координаты).

## Медиа

- Hero-видео: прямая ссылка на бесплатный клип с assets.mixkit.co (лицензия Mixkit, без
  атрибуции). Замените `components/Hero.tsx` → `<source src="...">` на собственное отснятое
  видео перед продакшном.
- Остальные изображения — прямые ссылки на Unsplash (`images.unsplash.com`). Замените на
  собственную фотографию экспедиций перед запуском.

## Что нужно донастроить перед продакшном

- Подключить реальную отправку формы (`components/Contact.tsx`, функция `handleSubmit`) —
  сейчас это просто визуальная имитация без бэкенда.
- Проставить реальные ссылки на Instagram/WhatsApp/Telegram в `components/Footer.tsx`.
- Заменить видео и фото на лицензированные материалы компании.
